Context

Architecture is the building block of the physical world and the environment we live in. Through Architecture we are able to explore different cultures and representations of how we view ourselves. How does technology play a part in Architecture? Technology is an expanding field and is influencing many industries. This website will look through the hardware and software technology innovations that already exist for Architecture, and pose possible inventions that’ll help in the Architecture field.

Software

Here are some software applications used in Architecture:
Sketchup

Sketchup allows you to prototype a design of sorts, which allows you to visualize your idea digitally.

RoomScan LiDAR/Pro

RoomScan Pro is used to get a floor plan of a room by drawing or scanning the room with your phone.

PictureThis

PictureThis identifies plants with just a picture. You can use this to see the plants around a landscape.

Cedreo

3D home design software that can be used to model houses with 2D or 3D floor plans, and realistic interior and exterior renderings.

Hardware

Here are some hardware applications used in Architecture:
Laser Measurer

A Laser Measurer measures a distance in a given space. The device emits laser light and measures the time it takes to get from one point to another to find the distance.

Scale Ruler

A scale ruler is used to draw measurements so that the design will be accurate. There are different measurement units on the ruler so it’s simple to use.

Drafting Board

A drafting board is a smooth board that is used to make drawings on paper so there aren't any bumps. It is also used to make precise measurements on your drawing.

Electric Eraser

An electric eraser helps with erasing specific lines or marks that are unwanted and is more efficient than a regular eraser when sketching.

Possible Inventions for Architecture

Software
Virtual Builder

This is software that will allow you to make 3d models of architectural structures and can be used to project them using light on actual land. This is extremely helpful if you want to first see how your model will look before actually building it.

Find It

A software that can identify the material used on something with just a picture. Just like how an app can identify what plant it is by a snap of a picture, “Find It” can be used when you want to use material on a building. When you’re unsure what the material is, you could use the app to help you.

Hardware
Pocket Sketch

The “Pocket Sketch” is essentially a foldable tablet that is used to draw sketches or make 3d models of a design. The tablet has a library that stores all your sketches and 3d models up to 128 GB. You can always add more space with an SD card. The foldable design makes it so that the tablet won't take up much space. Attached to the tablet is a stylus pen specifically designed for the tablet, which adds to the paper-like type of screen. Another useful tool is the scale ruler feature, which allows the user to draw accurate marks. There is a variety of colors to choose from, so the sketches won’t be in plain black and white.
